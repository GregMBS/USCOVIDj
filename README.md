# US State COVID-19 Graphs (USCOVIDj)

**USCOVIDj** is a web-based application that visualizes COVID-19 data for United States territories and states. Developed by Gregory M. Blumenthal at GMBS Consulting, this tool utilizes public data to generate interactive charts and graphs using JavaScript.

## üìä Features

* **Interactive Graphs:** Visualizes COVID-19 trends across different US states.
* **Real-time Data Fetching:** Pulls the latest available data directly from The New York Times public dataset.
* **Lightweight:** Built using vanilla JavaScript and HTML without the need for heavy backend frameworks.

## üöÄ Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

You need a modern web browser (Chrome, Firefox, Safari, Edge) to run the application. No Node.js or backend server is required.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/GregMBS/USCOVIDj.git](https://github.com/GregMBS/USCOVIDj.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd USCOVIDj
    ```
3.  **Run the Application:**
    * Simply open the `index.html` file in your web browser.

The application aggregates data from the following sources:
